## 项目用到的开源类库、组件
1. AFNetworking                         网络请求
2. Masonry                              轻量级的布局框架
3. FMDB                                 轻量级的数据库
4. KINWebBrowser                        Web 浏览器模块
5. MBProgressHUD                        显示提示或加载进度
6. SDWebImage                           加载网络图片和缓存图片
7. SSKeychain                           账号密码的存取
8. TTTAttributedLabel                   支持富文本显示的label
